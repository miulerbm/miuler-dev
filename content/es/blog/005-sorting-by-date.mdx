# Código

Construimos una función auxiliar para ordenar las Publicaciones que tienen una propiedad de fecha.

```js
export function sortPosts(posts: Array<Post>) {
  return posts.sort((a, b) => {
    if (a.date > b.date) return -1;
    if (a.date < b.date) return 1;
    return 0;
  });
}
```

# Explicación

Lo que hace esto es usar el método sort de JavaScript, el cual utiliza una función de devolución de llamada que devuelve valores lógicos a partir de una comparación entre dos elementos. En resumen, si a.date < b.date es verdadero, devuelve un -1, por lo que a.date es más reciente y a debe aparecer antes que b. Si a.date es menor que b.date, devuelve un 1, por lo que b es más reciente y debe aparecer antes que a.

<Callout type="message">
  Cuando la comparación devuelve un negativo, significa que “a”, el primer
  elemento, debe aparecer antes del segundo “b”. Y si la comparación devuelve un
  positivo, “a” debe aparecer después de “b”.
</Callout>

# Explicación Adicional

Otro ejemplo, ordenando números aleatorios:

```typescript
export function sortNumbers(numbers: number[]) {
  return numbers.sort((a, b) => a - b);
}
```

En esta función:

- `numbers` es un array de números que quieres ordenar.
- El método `sort` se utiliza en el array `numbers`.
- Dentro del método `sort`, hay una función de devolución de llamada que compara dos números `a` y `b`. Si `a` debería venir antes que `b` (es decir, `a` es menor que `b`), la función devuelve un valor negativo. Si `a` debería venir después que `b` (es decir, `a` es mayor que `b`), la función devuelve un valor positivo. Si `a` y `b` son iguales, la función devuelve `0`.
- Al restar `b` de `a` (`a - b`), estás ordenando los números en orden ascendente.

Puedes usar esta función para ordenar un array de números aleatorios así:

```typescript
const numbers = [5, 2, 8, 1, 9, 3, 6];
console.log(sortNumbers(numbers)); // Output: [1, 2, 3, 5, 6, 8, 9]
```

Esto mostrará el array ordenado `[1, 2, 3, 5, 6, 8, 9]`, donde los números están en orden ascendente.
